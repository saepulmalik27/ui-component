## React Component for rendering profile image

A basic div that renders profile image

### Component usage
```js
<Section styled={{}} decoration={[]}> children :  your html doc </Section>
```

### Property lists
| name  | description |
| ------------- | ------------- |
| decoration  | Array of Object terdiri dari style dari decoration, url image dan posisi dari decorasi tersebut |
| styled| styled component seperti background, shadow box dll|
| children| html tag sebagai section content|
